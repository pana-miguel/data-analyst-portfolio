# 📊 Data Analytics Portfolio  

## 🎯 Objetivo del repositorio
Este repositorio tiene como propósito demostrar habilidades en la parte de analista de datos mas especificamente en:

- Análisis exploratorio de datos (EDA)
- Limpieza y transformación de datos (ETL)
- Generación de insights
- Visualización de datos y metricas
- Construcción de dashboards

---

## 🧰 Tecnologías utilizadas

### 🔹 Procesamiento de datos
- Python
- Pandas
- Polars
- NumPy
- matplotlib 

### 🔹 Análisis y visualización
- Matplotlib
- Power BI
- Looker Studio

### 🔹 Entornos de trabajo
- Jupyter Notebook

### 🔹 Otras herramientas
- Google Cloud Platform (uso básico para almacenamiento de datos en algunos proyectos)

## 📁 Manejo de datos

Debido al tamaño de algunos datasets utilizados:

- No todos los datos están incluidos en el repositorio
- Se proporcionan versiones reducidas en `data/sample/`
- Los datasets completos seran referenciados externamente dentro de cada proyecto

Esto permite mantener el repositorio liviano y facilitar la reproducción de los análisis.

4. Visualización de resultados
