# 📊 Data Analytics Portfolio  

## ⚙️ Configuración del entorno y uso del repositorio

### 📥 Clonar el repositorio

Para trabajar con este proyecto en tu entorno local, primero clona el repositorio:

```bash
git clone https://github.com/pana-miguel/data-analyst-portfolio.git
cd data-analyst-portfolio
```

## 🐍 Crear entorno con Conda

Este proyecto incluye un archivo **environment.yml** con las herramientas nesesarias para el uso de todos los archivos.
ma explicaicon acerca de el uo de conda y para que irve [click aqui](./uso-de-herramientas/comandos_conda.md)

### 1. Crear el entorno de conda:
```bash
conda env create -f environment.yml
```
### 2. Activar el entorno:
```bash
conda activate datos
```
### 3. verificar instalacion:
```bash
python --version
```

## 📁 Estructura del repositorio

El repositorio está organizado por áreas de trabajo para reflejar las diferentes etapas del análisis de datos y el uso de herramientas:


### 📌 Descripción de cada sección

1. **Visualizacion/**  
  Contiene trabajos enfocados en la creación de gráficos y dashboards utilizando herramientas como Power BI, Looker Studio y Matplotlib.

2. **Procesos ETL/**  
  Incluye procesos de extracción, limpieza y transformación de datos utilizando principalmente Python, Pandas y Polars.

3. **Analisis/**  
  Se enfoca en el análisis exploratorio de datos (EDA), identificación de patrones y generación de insights.

4. **Herramientas/**  
  Espacio dedicado a la práctica y aprendizaje del uso de diferentes tecnologías, librerías y servicios relacionados con el análisis de datos.
  - [guia de comando de conda](./herramientas-comandos/comandos-conda.md)
  - [guia del uso de jupyter notebooks](./herramientas-comandos/comandos-jupyter.ipynb)
  - [guia del uso de numpy](./herramientas-comandos/comandos-numpy.ipynb)
  - [guia de pandas](./herramientas-comandos/comandos-pandas.ipynb)
  - [guia de polars](./herramientas-comandos/comandos-polars.ipynb)
  - [guia del uso y creacion de matrices](./herramientas-comandos/ejemplos-matrices-numpy.ipynb)
  - [funciones o consultas DAX en powerBI](./herramientas-comandos/funciones-dax-powerbi.md)
  - [guia de comandos de git y github](./herramientas-comandos/git-github.md)

5. **proyectos/**    
  Contiene proyectos completos donde se integran todas las etapas del proceso de datos: ETL, análisis y visualización, aplicados a casos más cercanos a escenarios reales.

## 📁 Manejo de datos

Debido al tamaño de algunos datasets utilizados:

- No todos los datos están incluidos en el repositorio
- Se proporcionan versiones reducidas en cada uno de los entorno donde se neseciten hacer el analissi de datos pero hay un dirve en el cual la referencia de descarga estara dentro de cada proyecto corresondientemente a los datos analizados en cada archivo
- Los datasets completos seran referenciados externamente dentro de cada proyecto

Esto permite mantener el repositorio liviano y facilitar la reproducción de los análisis.
