# 📈 Análisis de Datos

El análisis de datos consiste en explorar, interpretar y visualizar información con el objetivo de obtener insights que permitan comprender mejor una situación y apoyar la toma de decisiones.

A través del análisis, los datos pasan de ser simples registros a convertirse en información valiosa con significado y contexto.

## 🔍 Tipos de análisis:
- Descriptivo: ¿Qué pasó?
- Diagnóstico: ¿Por qué pasó?
- Predictivo: ¿Qué podría pasar?
- Prescriptivo: ¿Qué deberíamos hacer?

## 🛠️ Herramientas comunes

- Python
- Pandas
- Polars
- NumPy
- Matplotlib (para la generación de insights a través de visualizaciones)
- csv

## 🎯 Objetivo

El objetivo principal es comprender a profundidad los datos que se nos proporcionan, entender el contexto en el que fueron generados y definir qué acciones o análisis se pueden realizar a partir de ellos para generar valor estratégico.

Con esta información, es posible:

- Identificar patrones ocultos en los datos
- Analizar el crecimiento o decrecimiento de ciertas variables a lo largo del tiempo
- Interpretar tendencias y realizar aproximaciones o predicciones a futuro
- Detectar cambios relevantes que no son evidentes a simple vista
- Identificar anomalías o comportamientos atípicos
- Segmentar información para entender mejor distintos grupos o categorías
- Optimizar procesos basados en evidencia
- Apoyar la toma de decisiones con datos concretos
- Generar métricas e indicadores clave (KPIs)
- Facilitar la construcción de dashboards e informes interactivos
- Validar hipótesis de negocio o investigación
- Mejorar la calidad de los datos mediante su análisis
