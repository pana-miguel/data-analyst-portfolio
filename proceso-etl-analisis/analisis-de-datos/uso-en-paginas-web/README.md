# Importancia de los Frameworks Web para Mostrar Información

## Introducción

Hoy en día, la forma en que mostramos la información es tan importante como los datos mismos. Los frameworks web permiten transformar datos en experiencias visuales accesibles, interactivas y útiles para los usuarios con este acceso podemos mostrar tanto datos que ya pasaron o incluso graficas con datos en una base de datos que esta en constante alimentaicon de datos para mostrar diferentes metricas.

---

## ¿Por qué es importante mostrar información en la web?

* Acceso desde cualquier lugar
* Interacción en tiempo real
* Visualización clara e interactiva con los datos
* Escalabilidad
* Integración con múltiples sistemas para la interpretacion de la informacion

---

## FastAPI (Backend)

### ¿Qué es?

FastAPI es un framework moderno de Python para construir APIs rápidas y eficientes.

### Importancia

* Permite exponer datos fácilmente mediante endpoints
* Alto rendimiento (comparable con Node.js)
* Es muy utilizada para exponer datos de forma interactiva con la base de datos o alguna fuente de informacion que tengamos y lo que podemos hacer es consultas como esta en el ejemplo de el proyecto [buscar una cedula por medio de fastapi](./busqueda-con-fastapi/).

### Ejemplo de uso

* APIs para consultar bases de datos
* Servicios de búsqueda
* Microservicios

## Angular 

en algunos proyectos que se mostraran contienen el frontend con angular para el mejoramiento del rendimiento de carga de la pagina web

### ¿Qué es?

Angular es un framework de desarrollo frontend basado en TypeScript para crear aplicaciones web dinámicas.

### Importancia

* Permite construir interfaces modernas y dinámicas
* Manejo de componentes reutilizables
* Comunicación con APIs (como FastAPI)
* Escalabilidad para aplicaciones grandes

### Ejemplo de uso

* Dashboards
* Aplicaciones empresariales
* Sistemas administrativos
